Navbar 컴포넌트 상세 설명 React + React Router

파일 위치: src/components/Navbar.jsx · 본 문서는 현재 저장소의 최신 Navbar 구현을 기준으로 작성되었습니다.

핵심 역할

  • 라우팅 내비게이션: react-router-dom의 NavLink로 페이지 이동
  • 활성 링크 스타일링: isActive 상태로 현재 경로를 시각적으로 표시
  • 반응형 메뉴: 데스크톱 메뉴와 모바일 드로어 메뉴 제공

사용 라이브러리

  • react-router-dom: NavLink로 라우트 전환
  • react-icons: 햄버거(FiMenu), 닫기(FiX) 아이콘
  • 로컬 CSS: src/components/navbar.css

컴포넌트 코드

import React from "react";
import { useState } from "react";
import { NavLink } from "react-router-dom";
import { FiMenu, FiX } from "react-icons/fi";
import "./navbar.css";

export default function Navbar() {
  const [isMenuOpen, setIsMenuOpen] = useState(false);

  const navItems = [
    { to: "/", label: "Home" },
    { to: "/about", label: "About" },
    { to: "/skills", label: "Skills" },
    { to: "/projects", label: "Projects" },
    { to: "/contact", label: "Contact" },
  ];

  return (
    <nav className="navbar">
      <div className="navbar-container">
        <div className="navbar-content">
          <div className="navbar-logo">
            <span className="logo-text">Portfolio</span>
          </div>

          {/* Desktop Navigation */}
          <div className="desktop-nav">
            {navItems.map((item) => (
              <NavLink
                key={item.to}
                to={item.to}
                onClick={() => setIsMenuOpen(false)}
                className={({ isActive }) => `nav-button ${isActive ? "active" : ""}`}
              >
                {item.label}
              </NavLink>
            ))}
          </div>

          {/* Mobile menu button */}
          <button
            onClick={() => setIsMenuOpen(!isMenuOpen)}
            className="mobile-menu-button"
            aria-label="Toggle menu"
          >
            {isMenuOpen ? <FiX size={24} /> : <FiMenu size={24} />}
          </button>
        </div>

        {/* Mobile Navigation */}
        {isMenuOpen && (
          <div className="mobile-nav">
            <div className="mobile-nav-items">
              {navItems.map((item) => (
                <NavLink
                  key={item.to}
                  to={item.to}
                  onClick={() => setIsMenuOpen(false)}
                  className={({ isActive }) => `mobile-nav-button ${isActive ? "active" : ""}`}
                >
                  {item.label}
                </NavLink>
              ))}
            </div>
          </div>
        )}
      </div>
    </nav>
  );
}

위 코드는 JSX를 HTML에 안전하게 표시하기 위해 특수문자를 이스케이프했습니다.

구조와 동작

1) 상태 관리

  • isMenuOpen: 모바일 메뉴 열림/닫힘 상태를 관리합니다.
  • 모바일에서 링크 클릭 시 setIsMenuOpen(false)로 자동 닫힙니다.

2) 라우팅/활성화

  • NavLink 사용: className={({ isActive }) => ...}로 활성 스타일 부여
  • 활성 경로는 CSS에서 .nav-button.active, .mobile-nav-button.active 등으로 스타일링합니다.

3) 반응형 레이아웃

  • 데스크톱: .desktop-nav에 수평 메뉴
  • 모바일: 햄버거 버튼 클릭 시 .mobile-nav 표시

CSS 클래스 훅

  • .navbar, .navbar-container, .navbar-content
  • .navbar-logo, .logo-text
  • .desktop-nav, .nav-button, .nav-button.active
  • .mobile-menu-button
  • .mobile-nav, .mobile-nav-items, .mobile-nav-button, .mobile-nav-button.active

접근성 고려

  • 모바일 토글 버튼에 aria-label="Toggle menu" 적용
  • 아이콘은 시각적 보조. 버튼 텍스트 대체는 라벨로 처리

사용 방법

  1. react-router-dom 설치: npm i react-router-dom
  2. 앱 루트(src/main.jsx)를 <BrowserRouter>로 감쌉니다.
  3. src/App.jsx에 Routes/Route를 정의합니다.
  4. <Navbar />를 레이아웃 상단에 배치합니다.

확장 아이디어

  • 활성 경로 하이라이트에 애니메이션 바(underline) 적용
  • 드롭다운/메가메뉴 추가
  • 다국어(i18n) 라벨 변경
  • 권한/로그인 상태에 따른 메뉴 항목 제어
