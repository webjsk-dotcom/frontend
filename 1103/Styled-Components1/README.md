# Styled Components 튜토리얼

Styled Components를 배우기 위한 학습용 프로젝트입니다.

## 설치 방법

```bash
npm install
```

## 실행 방법

```bash
npm run dev
```

브라우저에서 `http://localhost:5173`을 열어 확인할 수 있습니다.

## 포함된 내용

이 프로젝트는 다음 Styled Components 기능들을 보여줍니다:

1. **기본 사용법** - styled 함수를 사용한 기본 스타일 컴포넌트 생성
2. **Props를 이용한 조건부 스타일** - props를 기반으로 동적 스타일 적용
3. **애니메이션** - keyframes를 사용한 애니메이션 효과
4. **테마 사용하기** - ThemeProvider를 통한 전역 테마 관리
5. **반응형 디자인** - 미디어 쿼리를 활용한 반응형 레이아웃
6. **중첩 스타일링** - 자식 요소에 대한 스타일 중첩

## 기술 스택

- React 18
- Styled Components 6
- Vite
- React Router DOM

## 학습 포인트

- `styled.div`, `styled.button` 등의 기본 문법
- Template literals을 사용한 스타일 정의
- ThemeProvider와 테마 객체 사용
- Props를 통한 조건부 스타일링
- keyframes를 이용한 애니메이션
- 중첩된 CSS 선택자
- 반응형 미디어 쿼리
