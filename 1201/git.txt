강제로  
$ git reset --hard HEAD~1
HEAD~1  전꺼 강제로 리셋(하나전꺼) 지워짐 vs에서만 git 원본남음

$ git push origin main --force  강제업데이트(git원본까지 삭제)

origin  -> git이다



$ git merge sub  ->  삭제됬던 vs 다시 올라옴
내컴에는 살리고 싶다.

$ git push origin main   -> git 에 다시 살려 올리겠다
