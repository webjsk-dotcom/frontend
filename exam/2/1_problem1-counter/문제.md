# 문제 1: 카운터 앱

## 난이도: ⭐ (기초)

## 요구사항

숫자를 증가/감소시킬 수 있는 간단한 카운터 애플리케이션을 만드세요.

### 기능 명세

1. **숫자 표시**
   - 화면에 현재 카운터 값을 크게 표시
   - 초기값은 0

2. **증가 버튼**
   - 클릭 시 카운터 값이 1 증가

3. **감소 버튼**
   - 클릭 시 카운터 값이 1 감소

4. **초기화 버튼**
   - 클릭 시 카운터 값이 0으로 리셋

## 제약사항

- React Hooks(useState)를 사용해야 합니다
- 클래스 컴포넌트는 사용하지 않습니다

## 예상 화면

```
        카운터: 5

    [증가]  [감소]  [초기화]
```

## 힌트

- `useState` 훅을 사용하여 카운터 상태를 관리하세요
- 각 버튼에 onClick 이벤트 핸들러를 연결하세요
- 상태 업데이트 함수를 호출하여 값을 변경하세요

## 보너스 (선택사항)

- 카운터 값이 양수일 때는 파란색, 음수일 때는 빨간색으로 표시
- 카운터 값이 0 미만으로 내려가지 않도록 제한
